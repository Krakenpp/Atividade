## PERGUNTAS SOBRE HTML

O que significa HTML?

HyperText Markup Language, uma linguaguem que ultilizar tags para marcar o conteudo

2. O que são TAGS em HTML?

Tags sao blocos de construçao elas falam onde cada parte do conteudo fica e como deve ser exibida

3. Como define uma TAG no HTML?

para definir uma tag, deve-se seguir a estrutura padrao ex:< P > para abertura e < /p > para fechamento, e dentro dessa estrutura deve-se colocar o conteudo

4. O que são atributos em HTML?

atributos sao informacoes extras que sao colocadas dentro de uma tag, geralmente para personalizar ou para atribuir uma configuracao

5. Qual a estrutura básica em HTML?

## PERGUNTAS SOBRE CSS

1. O que significa CSS?

Cascading Style Sheets, uma linguagem para dar uma identidade visual para a pagina

2. O que é SELETOR em CSS?

Um seletor e a parte do codigo que especifica qual elemento HTML ser aultilizado

3. Como define quais são os SELETORES no CSS?

e necessario identificar o elemento html e colocar a tag no codigo css e assim ultiliza-la como bem entender

4. Qual a estrutura básica em CSS?

seletor { propriedade: valor; }

5. Qual a hierarquia de prioridade nos seletores em CSS?

na ordem de mais fraco para o mais forte: Tags universais e combinadores; Classes, atributos e pseudo classes; Ids; estilo inline; Regra !importart
